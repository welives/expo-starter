# expo-starter

一个开箱即用的基于 `Expo` + `ReactNavigation` + `Zustand` + `MMKV` + `TailwindCSS` + `TypeScript` 的项目模板

这个工程的搭建笔记可以[在这里查看](https://welives.github.io/blog/front-end/engineering/expo.html)

## 使用

```sh
pnpm install
```

### 开发模式

```sh
pnpm start
# or
pnpm dev
```
